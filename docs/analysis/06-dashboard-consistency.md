## ממצאים עיקריים

### 3 רמות של דשבורדים:

**1. דשבורדים מלאים (6 תצוגות):**
שכר, דוחות שכר, מיסים, שירותים נלווים — עקביים ביניהם, עשירים

**2. דשבורדים חלקיים:**
- **ניהול** — חסר Gantt + חוברת עבודה + בחירת חודש
- **תוצאות כספיות** — חסר כמעט הכל חוץ מטבלה + Miro

**3. דשבורד ייחודי:**
- **לקוחות** — מבנה מטריצה שונה לחלוטין, בכוונה (spreadsheet-style)

### חסר בכל הדשבורדים:
- **כפתור הזרקת משימות** — אף דשבורד לא מאפשר הזרקה ישירה (צריך ללכת לעמוד נפרד)
- **סינון עומס קוגניטיבי** — רק שכר ומיסים, חסר בשאר
