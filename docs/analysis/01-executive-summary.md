# CalmPlan — סיכום מנהלים | ניתוח מערכת תכנון

**תאריך:** 7 באפריל 2026  
**צוות:** מהנדסים, מעצבים, פסיכולוגים — צוות CalmPlan

---

## תמונת מצב בקצרה

CalmPlan היא מערכת תכנון זמן ומשימות שנבנתה עבור משרד הנהלת חשבונות עצמאי, עם התאמות ל-ADHD. המערכת כוללת **58 עמודים**, **200+ קומפוננטות**, **12 מנועים עסקיים**, ומבוססת על React + Supabase + Electron.

**מה עובד טוב:** הליבה העסקית חזקה — עצי תהליכים, הזרקת משימות חוזרות, מכונת מצבים למיסים ושכר, ותצוגות ויזואליות בסגנון AYOA. מנוע ה-Cascade, ה-FocusMapView, ו-BadDayMode פועלים ומחוברים.

**מה לא עובד:** יש רכיבי ADHD שנבנו אך מנותקים (SmartNudge, MoodTracker), השעון הביולוגי מציג אך לא מסנן, סינון אנרגיה חסר ממשק, ועמודים רבים הם "מגילות אסתר" — רשימות ארוכות ללא קיפול, עימוד או ארגון. מנוע העיצוב מחובר רק ל-MindMap ולא למערכת השוטפת.

**השורה התחתונה:** המערכת עשירה במידע ובפונקציונליות, אבל חסר חיבור בין החלקים. צריך ליטוש, חיבור רכיבים מנותקים, וטיפול בחוויית משתמש — כדי שהמערכת תעבוד *בשביל* בעלת המשרד ולא תהפוך לעוד מקור עומס.

## טבלת מצב — כל הרכיבים

### רכיבים שעובדים (✅)

| רכיב | סטטוס | הסבר פשוט |
|-------|--------|------------|
| עצי תהליכים (Process Trees) | ✅ עובד | העץ שמגדיר איזה שירותים כל לקוח מקבל — עובד מלא |
| הזרקת משימות חוזרות | ✅ עובד | יצירת משימות חודשיות לפי העץ — עובד (ידני, לא אוטומטי) |
| מנוע Cascade (שרשרת משימות) | ✅ עובד | כשמשימה מסתיימת, הבאה נפתחת אוטומטית |
| FocusMapView (מה לעשות עכשיו) | ✅ עובד | מפת בועות שמראה רק משימות שאפשר לעשות — מחוברת ל-3 עמודים |
| BadDayMode (יום קשה) | ✅ עובד | כפתור שדוחה משימות לא-דחופות למחר |
| TimeAwareness (מודעות זמנים) | ✅ עובד | מראה דדליינים ודיווחים קרובים בכותרת העליונה |
| מערכת AYOA (מפות חשיבה) | ✅ עובד | 18 קומפוננטות ויזואליות — מפות רדיאליות, ענפים, צמתים |
| MindMap | ✅ עובד | תצוגת מפת חשיבה לפרויקטים והגדרות |
| כרטיסי לקוחות | ✅ עובד | הצגת מידע לקוח, שירותים, סטטוס |
| תזכורות (ReminderSystem) | ✅ עובד | תזכורות Toast כל 60 שניות — אירועים, משימות, סיכום |
| ניהול פרויקטים | ✅ עובד | CRUD מלא, שלבים, ציר זמן, מפת חשיבה |
| אוטומציות (5 מודולים) | ✅ עובד | שרשרת, ארכיון, סנכרון סטטוס, עומס קוגניטיבי, הזרקה |

### רכיבים שעובדים חלקית (⚠️)

| רכיב | סטטוס | מה חסר |
|-------|--------|--------|
| שעון ביולוגי | ⚠️ חלקי | מוצג בכותרת, אבל **לא מסנן משימות** לפי שעה ביום |
| סינון אנרגיה | ⚠️ חלקי | הלוגיקה קיימת ב-AppContext, אבל **אין ממשק** לבחירת רמת אנרגיה |
| מנוע עיצוב (DesignPanel) | ⚠️ חלקי | עובד ב-MindMap בלבד, **לא מחובר** לדשבורדים, טבלאות, קנבן |
| Cognitive Load (עומס קוגניטיבי) | ⚠️ חלקי | מוצג בדשבורדים, אבל **המגבלה לא נאכפת** (480 דקות מוגדר אך לא חוסם) |
| תצוגת Miro | ⚠️ חלקי | נוסף ל-6 דשבורדים אך שילוב מינימלי |

### רכיבים שלא מחוברים (❌)

| רכיב | סטטוס | מה המצב |
|-------|--------|---------|
